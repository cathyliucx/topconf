# TopConf 1.0.0

Initial public release.

Highlights:
- Native macOS menu-bar conference deadline tracker
- Calendar menu-bar icon
- Option-Space floating launcher
- CCF category and A/B/C rank discovery
- Up to 10 tracked conferences
- Correct multi-edition and timezone-aware deadline resolution
- Beijing-time deadline display
- Local reminder scheduling
- Structured upstream synchronization
- Offline persisted-catalog fallback
- Light and dark appearance support

TopConf requires macOS 14 or later.

Distribution note: this build is signed to run locally by Xcode. It is not notarized and is not distributed with a Developer ID signature in the current repository configuration.
